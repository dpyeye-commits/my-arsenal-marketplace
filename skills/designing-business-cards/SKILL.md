# Business Card SVG Generator

## 0단계 사전 확인 (필수)
- 회사·기관명
- 직군 (11 매트릭스 — 의료·법무·교육·디자인·IT·B2B 영업·예술·금융·NGO·정부·자영업)
- 이름·직책·연락처·주소
- 양면 vs 단면
- 후가공 (박·엠보·UV·레터프레스·재질 등)

## 1단계 — 규격 확정
- 표준 90×50mm (한국)
- Bleed 3mm 추가 → 96×56mm
- Safe zone 3mm 안쪽

## 2단계 — 직군별 톤 매핑
- 의료: 신뢰·청결 (네이비·화이트)
- 법무: 권위 (블랙·골드)
- 교육: 따뜻 (베이지·녹)
- 디자인: 개성 (자유)
- (11종 매트릭스)

## 3단계 — 타이포그래피 황금비
- 회사명 : 이름 : 직책 = 1 : 1.4 : 0.7
- 자간 (회사명 -3, 본문 -1.2)
- 줄간 1.2 (좁힘)

## 4단계 — SVG 생성
- 앞면: 로고·회사명·이름·직책·연락처
- 뒷면: 슬로건·QR·NFC 태그(선택)
- CMYK 변환 가이드 주석

## 5단계 — 후가공 가이드
- 박(금·은·홀로) / 엠보(양각) / 디보스(음각) / UV / 레터프레스
- 특수 소재 (씨앗종이·아크릴·메탈)

## 추측 금지
- 사용자 미입력 정보(전화번호 끝자리·팩스) 임의 생성 X
