# Inflearn Course Builder (12장 풀스택)

## 0단계 사전 확인
- 강의 주제·핵심 1줄
- 대상 (입문/중급/실무)
- 강의 길이 (시간)
- 가격대 (₩44K~₩99K)

## 1단계 — PROPOSAL.md 12장 골격
- 1장 도구 환경 / 2장 핵심 개념 / 3~10장 핵심 콘텐츠 / 11장 양산 운영 / 12장 수익화
- 각 장 학습 목표 3개 + 실습 1개

## 2단계 — 슬라이드 자동 양산
- Marp (Markdown → PPTX)
- python-pptx 후처리
- Quarto revealjs (학술 톤)

## 3단계 — 실습 코드
- 각 장 실습 코드 1~3개
- GitHub 레포 동기화

## 4단계 — 부록
- 5종 인용 양식 변환 (APA·시카고·MLA·Harvard·Vancouver)
- Citation.js 자동화 코드

## 5단계 — 마케팅 자료
- 인프런 등록 메타 (제목·요약·키워드)
- 썸네일·예고편 영상 스크립트

## 추측 금지
